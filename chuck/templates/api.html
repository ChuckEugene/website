Finance API

Overview:

  • This API seeks to make acquiring financial data easier for analysis
  • S&P 500 data is scraped from yahoo finance daily and inserted into a SQL database
  • All requests, calculations, and updates are multiprocessed for efficiency

Data:

  • Company ticker and name
  • Price at close any day in the past 5 years
  • Percent returns for one day, one week, one month, three months, and six months

Parameters:

  • abbr(required) the company's ticker
  • t1 given in format YYYY-MM-DD, default value is 0
  • t2 default value is current date
  • d_type(optional) p, d, w, m, three, six for price and one day, one month, three month and six month returns

Traveling Salesman API

Overview:

  • The travelling salesman, an np-Hard problem, seeks to find the shortest route between several points
  • Coded in C++, my solution utilizes the 2-opt heuristic and is optimized for time and memory
  • 35 vertices from request to response takes under 3 seconds

Data:

  • A text file is uploaded with coordinates on a graph
  • The total distance of the optimal route is returned along with the indices of the coordinates that are connected

Parameters:

  • A single text file attached to a request to /api/TSP/
  • Format is number of vertices on first line, then x y coordinates on every sequential line
