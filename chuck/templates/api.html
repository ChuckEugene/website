Finance API

Data: stock prices and forward returns for S&P 500 companies spanning the past 5 years

Parameters:

  • abbr(required) the company's ticker
  • t1(optional, default = 0) the start date for the span of the data. date format (YYYY-MM-DD)
  • t2(optional, default = current date) the end date for the span of the data
  • d_type(optional, default = price) prices, and percent returns from one day, one week, one month, 3 months and six months
